**JavaScript -- PixelGrid class:**
```javascript
class PixelGrid {
  constructor(size) {
    this.size = size;
    this.cells = new Array(size * size).fill(0); // 0=unset, 1=white, 2=black
  }
  get(x, y) { return this.cells[y * this.size + x]; }
  set(x, y, state) { this.cells[y * this.size + x] = state; }
  cycle(x, y) { this.set(x, y, (this.get(x, y) + 1) % 3); }
  clear() { this.cells.fill(0); }
}
```

**JavaScript -- Canvas rendering:**
- Create a `renderPaintGrid(canvas, grid)` function that:

    - 1 (white locked): `#ffffff` (pure white)
