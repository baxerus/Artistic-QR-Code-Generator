**New JS functions** (add before updateTopResults):

1. `renderResultToCanvas(moduleData, moduleCount, canvasSize)`:
   - Creates an offscreen canvas at canvasSize x canvasSize pixels
   - Calculates scale = Math.floor(canvasSize / moduleCount)
   - Fills white background first (ctx.fillStyle = '#FFFFFF', fillRect full canvas)
   - Then iterates moduleData, drawing black modules at scale
   - Returns the canvas element
   - Use fillRect per module (not ImageData) so the exported PNG has clean scaled pixels

2. `downloadCanvasAsPNG(canvas, filename)`:
   - Validates canvas exists and has non-zero dimensions
   - Calls canvas.toBlob() with 'image/png' type
   - In callback: checks blob is not null, creates URL.createObjectURL(blob), creates anchor element with href=url and download=filename+'.png', appends to body, clicks, removes from body, calls URL.revokeObjectURL(url)
   - On null blob: console.error and return silently

3. `copyToClipboard(text, button)`:
   - Async function
   - First check: if navigator.clipboard is not available (file:// protocol), fall back to creating a temporary textarea element, setting its value, selecting, calling document.execCommand('copy'), then removing it. Show feedback on button regardless of method.
   - If clipboard API available: await navigator.clipboard.writeText(text)
   - On success: save button's original textContent, set to 'Copied!', disable button, setTimeout 2000ms to restore
   - On error: console.error, set button text to 'Failed' briefly, restore after 2000ms
   - IMPORTANT: The fallback for file:// is critical since Clipboard API requires secure context (HTTPS)
