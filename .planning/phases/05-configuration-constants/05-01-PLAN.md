  **B. Auto-bump with green glow animation (locked decision):**

  Modify `updateVersionDropdown(minVersion)` to:
  1. Capture the dropdown's current value BEFORE clearing innerHTML
  2. After populating options and setting value, check if the value changed
  3. If value changed AND there was a previous numeric value (not empty/placeholder), add the `result-highlight` class to the dropdown
  4. Remove the class after 600ms so the animation can replay on future bumps

  Critical pitfall: Must remove the animation class before re-adding so CSS animation can replay. Use this pattern:
  ```javascript
  dropdown.classList.remove('result-highlight');
  // Force reflow so browser recognizes removal
  void dropdown.offsetWidth;
  dropdown.classList.add('result-highlight');
  setTimeout(() => dropdown.classList.remove('result-highlight'), 600);
  ```

  **C. Auto-increase only, never auto-decrease (locked decision):**

  In `updateVersionDropdown(minVersion)`, after populating options:
  1. If the dropdown previously had a valid numeric value, and that value >= minVersion, keep the old value (do NOT decrease to minVersion)
  2. Only set dropdown.value = minVersion if the previous value was empty/placeholder OR if minVersion > previous value
  3. This means: if user had version 5 selected and URL gets shorter (min becomes 3), dropdown stays at 5. User can manually lower it.
  4. If user had version 5 and URL gets longer (min becomes 7), auto-bump to 7 with glow animation.
  5. The dropdown should still only show versions >= minVersion (hide smaller ones). If the user's current version is valid, keep it selected.

  The full updated updateVersionDropdown should look approximately like:
  ```javascript
  function updateVersionDropdown(minVersion) {
    const dropdown = document.getElementById("version-select");
    const previousValue = dropdown.value ? parseInt(dropdown.value) : null;
    dropdown.innerHTML = "";
    dropdown.disabled = false;

    for (let v = minVersion; v <= MAX_QR_VERSION; v++) {
      const option = document.createElement("option");
      option.value = v;
      option.textContent = "Version " + v + " (" + getModuleSize(v) + "\u00D7" + getModuleSize(v) + ")";
      dropdown.appendChild(option);
    }

    // Auto-increase only: keep previous version if still valid, bump if needed
    let selectedVersion = minVersion;
    if (previousValue !== null && !isNaN(previousValue)) {
      if (previousValue >= minVersion) {
        // Previous version still valid - keep it (never auto-decrease)
        selectedVersion = previousValue;
      } else {
        // Previous version too small - auto-bump to minimum
        selectedVersion = minVersion;
      }
    }

    dropdown.value = selectedVersion;

    // Green glow on auto-bump (only when version increased from a previous value)
    if (previousValue !== null && !isNaN(previousValue) && selectedVersion > previousValue) {
      dropdown.classList.remove('result-highlight');
      void dropdown.offsetWidth; // force reflow for animation replay
      dropdown.classList.add('result-highlight');
      setTimeout(() => dropdown.classList.remove('result-highlight'), 600);
    }
  }
  ```
