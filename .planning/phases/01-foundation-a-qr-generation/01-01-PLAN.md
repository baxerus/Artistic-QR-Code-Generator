**CSS Styling (locked decisions from user):**

Page layout:
- `body` background: light gray (#f5f5f5 or similar subtle gray)
- `.container` centered with max-width (~600px), auto margins
- Content centered on page vertically and horizontally

Typography (Claude's discretion):
- Font family: system-ui, -apple-system, sans-serif (system font stack)
- h1: ~1.5rem, medium weight, subtle color
- Body text: ~0.875rem-1rem

Header:
- Minimal -- tool name only "Artistic QR Code Generator"
- No description text
- Subtle bottom border or spacing to separate from form

Form controls:
- URL input: full width within container, decent padding (~10-12px), rounded corners
- Input wrapper: position relative (for validation icon positioning)
- Validation icon: positioned absolute inside input right side
- Valid state: green border (#10b981), checkmark icon
- Invalid state: red border (#ef4444), X icon
- Version dropdown: full width, same styling as input
- Label for version: small, subtle text above dropdown
- Generate button: full width, prominent style (blue/primary color), disabled state grayed out

Spacing:
- 20-30px vertical gaps between form groups (`.input-group`)
- Sections feel connected but distinct

Canvas container:
- Canvas: width 100%, max-width ~400-500px, centered
- Aspect ratio: 1/1 (square always)
- Border: 2px solid #e0e0e0 or similar subtle border
- Box shadow: 0 4px 12px rgba(0,0,0,0.1) or similar elevation
- Background: white (contrasts with gray page background)
- Margin top: 20-30px from form

Footer:
- Minimal credit/version text
- Small font, muted color
- Centered, bottom of container

Responsive (Claude's discretion):
- Container shrinks on small screens with padding
- Canvas scales down proportionally
- Touch-friendly input sizes
